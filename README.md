# Capstone 2: Business Analysis with Python  
## Sales Territory Analysis – Northeast Region

### Project Description
This project analyzes sales performance for **EmporiUm**, a fictional retail company that sells textbooks, technology products, art supplies, and other merchandise both in-store and online. The analysis focuses on comparing sales trends and performance across two assigned sales territories in the Northeast region.

Using Python, Pandas, and Matplotlib, this project explores sales data to identify patterns, evaluate product category performance, and generate data-driven marketing recommendations.

### Sales Territories Analyzed
- **Sales Territory 1:** Maine  
- **Sales Territory 2:** Massachusetts  

### Tools & Technologies
- Python  
- Jupyter Notebook  
- Pandas  
- Matplotlib  

### Key Analysis Areas
- Monthly revenue trends  
- Sales performance by territory and store  
- Customer purchasing behavior  
- Product category analysis  
- Marketing insights and recommendations  

### Project Files
- `monroy_sales_analysis.ipynb` – Jupyter Notebook containing the full data analysis and visualizations
